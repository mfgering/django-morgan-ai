About Morgan

Dawson Information: Morgan uses a set of files that are specific to The Dawson:

  • Bylaws
  • Covenants
  • Maintenance Responsibilities
  • Rules and Regulations
  • Parking, storage bin, and personal locker assignments
  • Unit size, common element percentages
  • Miscellaneous facts, e.g. about who manages The Dawson

Morgan will resist answering questions beyond the scope of The Dawson.

Morgan uses these instructions to guide it.

All Messages in a Chat: Morgan considers all the messages in a chat when it generates a response. This means that it will remember your name, if you tell it in a previous message. It also means you can ask for more or less detail without having to repeat the question, e.g. "Less detail, please."

Natural LanguageMorgan understands natural language very well. It knows, e.g., these are asking the same question:

  • What is the size of unit 412?
  • How big is unit 412?

Implementation: The smarts for Morgan are provided by OpenAI, using their Assistant API and files uploaded to the assistant. This enables OpenAI to run general tasks, but to use the files as additional sources of data.
